Rx Store

The stream management library

Get Started

High Performance

Only render/deliver updates for what changed, nothing more.

Reactive

The primitive source of truth are individual event emitters. Combine them to build "up" streams instead of "paring down" state (like with other state managers).

Functional

Abstract away the dimension of time, using the full power of RxJS. Declare complex async logic with pure functions.
